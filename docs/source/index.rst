Welcome to xtask's documentation!
=================================

**xtask** is an open-source Python library that provides a powerful toolkit of orchestrating execution of tasks within different directories of a project.

**xtask** aims to be a simple yet powerful and extensible build tool that is easy to setup, fast and easy to use, and highly portable.

.. note::
	This project is under active development.
